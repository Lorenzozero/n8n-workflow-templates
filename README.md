## 🚀 Quick Start

1. **Clona il repository**
   ```bash
   git clone https://github.com/Lorenzozero/n8n-workflow-templates.git
   ```

2. **Scegli il template**
   - Naviga nella cartella della categoria desiderata
   - Leggi il README specifico del template
   - Scarica il file JSON del workflow

3. **Importa in n8n**
   - Apri la tua istanza n8n
   - Vai su "Import from File" o "Import from URL"
   - Carica il file JSON del template
   - Configura le credenziali delle app

4. **Personalizza e attiva**
   - Modifica i parametri secondo le tue necessità
   - Testa il workflow in modalità debug
   - Attiva l'automazione
